[깜빡임 방지를 위한 더블 버퍼링 적용]

○ 플레이어 조작
-  상 하 좌 우 대각 8 방향으로 이동 및 구르기
-  구르기는 마우스 우클릭을 통하여 이루어지고 좌클릭은 총알 발포를 위해 사용 됨
-  플레이어는 이동에따라 그 외의 오브젝트들도 전부 움직여야 하기에 스크롤 적용
2일차
○ 플레이어 조작
- 플레이어가 총, 구르기 를 위한 마우스 위치에 따라 바라보는 방향이 다르기에 위치 값에 따른 이미지 변경 적용
  (기본적으로 플레이어는 마우스 기준으로 시선을 두고있음)
- 플레이어는 구르기중 무적 판정이기에 bool값을 통하여 무적 판단

○ 플레이어 무기
- 플레이어의 무기들은 손에 잡혀있는 형태로 되어 있아야함 
-    기본총 :  한발씩 총알이 나가는 형태
-       샷건 :  6개의 총알이 퍼져서 나가는 형태
- 로켓런처 :  미사일 모양의 총알이 발포되며 총알과 충돌하는 대상은 폭발 데미지
 		  기본 애니메이션 적용

3일차
○ 일반 몬스터
- 기사 : 갑옷을 입고있는 몬스터 대검을 휘두를시 휘두르는 방향으로 부채꼴 형태로 총알이 발쏴됨 
   (기본적으로 모든 몬스터는 일정 시간마다 플레이어를 추적)
- 총알 : 총알 형태의 몬스터 구현 가지고 있는 무기에 따른 총알 발포
- 슬라임 : 플레이어를 향해 돌진하여 몸통박치기
4일차
○ 일반 몬스터
- 메이지 : 영창 모션 이후 총알을 N, R, O 형태로 그려서 플레이어에게 발포

○ 보스 몬스터
- 패턴1 : 플레이어를 본 후 2개의 쿠나이 던지기 
- 패턴2 : 360도 1번 회전을 하면서 치즈 발포
  기본 애니메이션 적용
5일차
○ 보스 몬스터
- 패턴3 : 구현 보스 기준으로 팔방향에 치즈(총알)가 생긴 후 보스 주변에 케이크 형태로 조각이 모이게 되고 다 모이면 보스가 내려찍어서 
  터트린 후 모아놨던 치즈가 팔방향으로 퍼짐
- 패턴4 : 보스가 360도로 4번정도 회전하면서 일반 총알 발포
  기본 애니메이션 적용
6일차
○ 보스 몬스터
- 패턴5 부채꼴 방향으로 쿠나이를 빠르게 2번씩 던짐 총 3번 발동 후 양 옆으로 2번씩 추가로 던짐
  보스 사망 애니메이션 적용
○ 스테이지 (스테이지는 입장시 문이 닫히며 클리어 전까지 나갈 수 없게 구현 예정)
- 스테이지1 : 슬라임, 불릿 몬스터가 소환되게 적용 
- 스테이지2 : 스테이지는 불릿, 메이지 몬스터 소환
- 스테이지3 : 스테이지는 불릿, 메이지, 기사 몬스터 소환
- 스테이지4 : 스테이지는 보스 소환
  보스 스테이지를 제외하고 클리어시 보물상자 생성되게 할 예정
7일차
○ 충돌, UI, 사운드
- 객체간 충돌시 총알 사라지게 충돌 구현
- 스테이지 입장시 닫히는 문에 대하여 나갈 수 없게 충돌 구현
- 보스 HP UI 적용
- 사운드 적용
8일차
○ 오브젝트, UI
- 플레이어 UI 좌측 상단 HP, 열쇠 구현
- 열쇠 아이템 구현 보물상자와 상호 작용시 열쇠를 소모하여 오픈되며 아이템을 드랍
- 플레이어 UI 우측 하단 현재 들고있는 무기 이미지, 탄환이 어느정도 남았는지 보여주기 위해 구현
9일차
○ 오브젝트, UI
- 무기가 들어있는 보물상자 구현
- 플레이어 장전시 머리 위에 장전중이라는 애니메이션 구현
- 몬스터가 죽을시 보물상자를 열수있는 열쇠가 드랍되게 구현
  게임 중간점검
10일차
○ 오브젝트
- 플레이어가 열쇠를 들고있을경우 보물상자와 상호작용하여 열수있게 구현
- 스테이지 진입시 문이 닫히며 클리어전까지 나갈 수 없게 구현
- 플레이어와 상호작용하는 탁자 구현 탁자는 엎을 수 있으며 총알을 일정 데미지 막을 수 있다.
11일차
○ 오브젝트, UI
- 총알과 상호작용하는 폭발 오크통 구현 오크통은 총알에 닿을시 일정범위에 폭발을 일으킴
- 보스방 클리어시 총알모양의 엘리베이터 생성(엔딩 진입을 위함)
- 게임 Main 화면 제작 시작, 나가기 버튼 구현
12일차
○ 오브젝트, UI, BOSS
- 플레이어 시점 스크롤이 아닌 카메라로 변경하여 적용 해보기 
- 카메라가 적용되면 보스방 입장시 연출 추가 해보기 카메라가 보스쪽으로 천천히 이동되며 PLAYER와 보스 이미지가 출력됨
- 보스 패턴 임의로 하나 만들어서 추가 해보기
- 총알 모양 엘리베이터 진입시 엔딩 화면 출력
13일차
○ 오브젝트, 사운드, NPC, 몬스터
- 가능하면 NPC와 상호작용 하여 아이템을 구매/판매가 가능하게 추가 해보기
- 가능하면 Bird 몬스터 추가하기 입에서 알을 뱉고 그 알이 땅에 닿으면 터지면서 총알 여러발이 플레이어를 향해 발포
- 추가된 오브젝트의 사운드 적용
- 타일 매니저를 이용하여 배경 꾸며보기

14일차
○ 최종점검, 영상편집
- 게임 최종 점검 문제가 생기는 부분은 수정 가능하면 수정을 하고 수정을 못할시 해당부분은 제거
- 영상 편집/녹화
